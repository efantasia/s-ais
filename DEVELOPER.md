sAIS Developer
==============

The documentation intended for sAIS developers.

Install
-------

Clone the sAIS repo

```
git clone https://github.com/infinitedescent/sais.git
```

Navigate into the project directory

```
cd sais
```

Docker
------

Docker is utilized to run the a local PostGRES database.

To start Docker run:

```
docker-compose up -d 
```

To stop Docker run:

```
docker-compose down
```
